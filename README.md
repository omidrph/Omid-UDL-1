# 🚀 دانلودر خودکار فایل با GitHub Actions

یک ابزار قدرتمند برای دانلود فایل‌ها و ذخیره‌سازی دائمی در ریپازیتوری گیت‌هاب شما.

---

## ✨ قابلیت‌ها

| قابلیت | توضیحات |
|--------|----------|
| ⬇️ **دانلود از هر لینک مستقیم** | پشتیبانی از HTTP/HTTPS |
| 📦 **فایل‌های بزرگ** | تست شده تا ۲۸ گیگابایت |
| ✂️ **تقسیم خودکار** | فایل‌های بزرگ به چند قسمت ۴۵ مگابایتی |
| 🔒 **رمزگذاری** | قابلیت رمزگذاری فایل‌های ZIP |
| ⚡ **دانلود همزمان** | قابلیت دانلود چند فایل با هم |
| 📁 **ذخیره دائمی** | فایل‌ها در گیت‌هاب شما باقی می‌مانند |
| 🔄 **شکست خودکار** | در صورت خطا، دانلود مجدد می‌شود |
| 📊 **گزارش کامل** | README خودکار با لینک‌های دانلود |

---

## 🚀 نحوه استفاده

### 1️⃣ دانلود فایل

1. به تب **Actions** بروید
2. روی **"⬇️ دانلود فایل"** کلیک کنید
3. دکمه **"Run workflow"** را بزنید
4. اطلاعات زیر را وارد کنید:

| فیلد | توضیح | مثال |
|------|-------|------|
| **لینک‌های دانلود** | لینک مستقیم فایل(ها) | `https://example.com/file.zip` |
| **حالت دانلود** | `normal` یا `zip` | `normal` |
| **رمز عبور** | رمز برای فایل ZIP (اختیاری) | `mysecret123` |
| **حجم هر قسمت** | حداکثر حجم هر قسمت (MB) | `45` |

5. دوباره روی **"Run workflow"** کلیک کنید
6. منتظر بمانید تا دانلود کامل شود

### 2️⃣ مشاهده فایل‌های دانلود شده

- به پوشه [`downloads/`](./downloads) بروید
- هر فایل در یک پوشه جداگانه قرار دارد
- داخل هر پوشه یک `README.md` با لینک دانلود وجود دارد

### 3️⃣ پاک کردن فایل‌ها

1. به تب **Actions** بروید
2. روی **"🧹 پاک کردن دانلودها"** کلیک کنید
3. روی **"Run workflow"** کلیک کنید
4. در کادر `warning` کلمه `confirm` را بنویسید
5. دوباره روی **"Run workflow"** کلیک کنید

## 📖 راهنمای دقیق

### حالت‌های دانلود

| حالت | توضیح |
|------|-------|
| **normal** | هر فایل به صورت جداگانه در پوشه خودش ذخیره می‌شود |
| **zip** | تمام فایل‌ها در یک آرشیو ZIP قرار می‌گیرند |

### تقسیم خودکار فایل‌های بزرگ

- فایل‌های بالاتر از حجم تعیین شده (پیش‌فرض 45 مگابایت) به صورت خودکار تقسیم می‌شوند
- قسمت‌ها با نام‌های `.zip`، `.z01`، `.z02` و ... ذخیره می‌شوند
- برای extract کردن، همه قسمت‌ها را دانلود و فایل اصلی `.zip` را باز کنید

### آرشیو رمزگذاری شده

اگر رمز عبور وارد کنید:
- فایل ZIP با رمز عبور محافظت می‌شود
- برای extract کردن نیاز به وارد کردن رمز دارید

## ⚙️ تنظیمات پیشرفته

### تغییر حجم هر قسمت

در فیلد `split-size` می‌توانید حجم هر قسمت را تغییر دهید:
- حداقل: 10 مگابایت
- حداکثر: 100 مگابایت
- پیش‌فرض: 45 مگابایت (مناسب گیت‌هاب)

### دانلود همزمان

ابزار به صورت خودکار از دانلود همزمان با `aria2` استفاده می‌کند:
- ۴ اتصال همزمان به سرور
- تقسیم خودکار فایل به ۴ قسمت
- در صورت خطا، fallback به `curl`

## 🐛 رفع مشکل

### مشکل: دانلود انجام نمی‌شود

**راه‌حل:**
1. بررسی کنید لینک مستقیم و قابل دسترس باشد
2. برخی سرورها (مثل سافت‌۹۸) IP خارج را مسدود کرده‌اند
3. فایل ممکن است حذف شده باشد

### مشکل: آپلود در گیت‌هاب با خطا مواجه می‌شود

**راه‌حل:**
1. حجم فایل خیلی بزرگ است (بیشتر از ۲۸ گیگابایت)
2. اتصال اینترنت را بررسی کنید
3. دوباره تلاش کنید - سیستم به صورت خودکار retry می‌کند

### مشکل: فایل ZIP باز نمی‌شود

**راه‌حل:**
1. مطمئن شوید همه قسمت‌ها را دانلود کرده‌اید
2. اگر رمز دارد، رمز صحیح را وارد کنید
3. از 7-Zip یا WinRAR استفاده کنید

1. **فقط از ریپازیتوری شخصی خود استفاده کنید**
